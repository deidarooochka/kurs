Файл kursovaya.cpp - тут лежит код с тестированием алгоритмов
Программа для сравнительного анализа эффективности различных алгоритмов сортировки в наихудшем, среднем и наилучшем случаях.
1) Пузырьковая сортировка (bubble sort)
2) Сортировка слиянием (merge sort)
3) А также встроенная функция сортировки std::sort
4) и std::stable_sort
Наилучший случай:уже отсортированный массив 
Средний случай: случайные числа в диапазоне 
Наихудший случай: обратно отсортированный массив

Программа тестирует массивы следующих размеров: 10, 20, 50, 100, 200, 500, 1000, 2000, 5000, 10000 элементов
