Файл kursovaya/kursovaya.cpp - тут лежит код с реализацией и тестированием алгоритмов (весь код курсовой работы)
Программа для сравнительного анализа эффективности алгоритма сортировки пузырьком и слиянием (bubble sort и merge sort).
1) Пузырьковая сортировка (bubble sort)
2) Сортировка слиянием (merge sort)
3) А также встроенная функция сортировки std::sort

Программа тестирует массивы следующих размеров: 10, 20, 50, 100, 200, 500, 1000, 2000, 5000, 10000 элементов
